# 🔐 CyberSec & OSINT Hub

Portal educacional sobre Cybersegurança e OSINT (Open Source Intelligence).

## 🚀 Deploy na Vercel

1. Crie um repositório no GitHub com estes arquivos
2. Acesse [vercel.com](https://vercel.com)
3. Clique em **"New Project"** e importe o repositório
4. Clique em **"Deploy"** — pronto!

## 🛡️ Segurança Implementada

| Header | Proteção |
|--------|----------|
| **CSP** | Restringe origens de scripts, estilos e recursos |
| **HSTS** | Força HTTPS por 1 ano com preload |
| **X-Frame-Options** | Previne clickjacking (DENY) |
| **X-Content-Type-Options** | Previne MIME sniffing |
| **X-XSS-Protection** | Filtro XSS do navegador |
| **Referrer-Policy** | Controle de informações de referrer |
| **Permissions-Policy** | Bloqueia câmera, microfone, geolocalização |
| **COOP/CORP** | Isolamento de cross-origin |
