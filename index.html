01. Sobre Cybersegurança

🛡️

Defesa Cibernética

Proteção de sistemas, redes e dados contra ataques cibernéticos. Inclui firewalls, IDS/IPS, SIEM e resposta a incidentes.

  • Network Security
  • Endpoint Protection
  • Threat Intelligence
  • Incident Response
⚔️

Offensive Security

Testes de penetração, red teaming e avaliação de vulnerabilidades para fortalecer a postura de segurança.

  • Penetration Testing
  • Red Team Operations
  • Vulnerability Research
  • Social Engineering
🔍

Inteligência & OSINT

Coleta e análise de informações de fontes公开mente disponíveis para apoio à tomada de decisões de segurança.

  • Reconnaissance
  • Footprinting
  • Data Correlation
  • Threat Hunting
🔐

Criptografia

Proteção de dados através de técnicas criptográficas avançadas, incluindo cifragem, hash e assinaturas digitais.

  • Encryption (AES, RSA)
  • Hash Functions
  • Digital Signatures
  • PKI Infrastructure

# OSINT - Open Source Intelligence

Metodologia de coleta de inteligência a partir de fontes públicas.


Fases:

1. Planning → Definição de objetivos e escopo

2. Collection → Coleta de dados de fontes abertas

3. Processing → Organização e filtragem dos dados

4. Analysis → Correlação e análise de inteligência

5. Dissemination → Relatório e recomendações

6. Feedback → Refinamento do processo

04. Técnicas Avançadas

🕸️

Google Dorking

Uso avançado de operadores de busca do Google para encontrar informações sensíveis expostas.

site:target.com ext:pdf confidential inurl:admin intitle:"login" filetype:xls "password" site:gov.br intext:"index of /" "parent directory"
🌐

Subdomain Enumeration

Descoberta de subdomínios para expandir a superfície de ataque de um alvo.

subfinder -d target.com -all amass enum -d target.com assetfinder --subs-only target.com gau target.com | grep -v ".css\|.js"
📧

Email Reconnaissance

Identificação de emails corporativos, padrões de nomenclatura e verificação de vazamentos.

theHarvester -d target.com -b all hunter.io → email pattern verify emails with EmailRep.io check HaveIBeenPwned for breaches
🔎

Shodan Recon

Motor de busca para dispositivos conectados à internet. Encontra servidores, câmeras, IoT e mais.

shodan search "country:BR org:target" shodan host 1.2.3.4 port:443 ssl:"target.com" http.title:"dashboard" country:BR
📸

Image OSINT

Análise forense de imagens: metadados EXIF, geolocalização e reverse image search.

exiftool image.jpg exiftool -GPS* image.jpg reverse search: Google, Yandex, TinEye GeoSpy.ai → AI geolocation
🗃️

Archive Analysis

Análise de conteúdo histórico de websites através de archives e snapshots antigos.

waybackurls target.com web.archive.org/web/*/target.com gau --subs target.com waybackpy --url target.com

05. Boas Práticas de Segurança

01

Autenticação Forte

  • MFA/2FA em todas as contas
  • Hardware Security Keys (YubiKey)
  • Senhas únicas por serviço
  • Gerenciador de senhas (Bitwarden, 1Password)
02

Proteção de Rede

  • VPN para conexões públicas
  • DNS-over-HTTPS (DoH)
  • Firewall configurado
  • Segmentação de rede
03

Privacidade Digital

  • Navegador hardened (Firefox + uBlock)
  • Criptografia de disco completo
  • Comunicação E2E (Signal)
  • Limpeza regular de metadados
04

Hardening de Sistemas

  • Updates e patches em dia
  • Princípio do menor privilégio
  • Auditoria de logs (SIEM)
  • Backups 3-2-1 com imutabilidade
05

Segurança Web

  • Headers de segurança (CSP, HSTS)
  • WAF configurado
  • HTTPS/TLS 1.3 obrigatório
  • SAST/DAST no CI/CD
06

Conscientização

  • Treinamento anti-phishing
  • Política de senhas forte
  • Plano de resposta a incidentes
  • Simulações regulares de ataque
